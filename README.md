## Cài đặt môi trường phát triển

Để bắt đầu phát triển ứng dụng ReactJS, bạn cần cài đặt VScode (Visual Studio Code) và Node.js và npm (Node Package Manager).

-   Bạn có thể tải VScode từ trang chính thức: [https://code.visualstudio.com/](https://code.visualstudio.com/).
-   Bạn có thể tải Node.js từ trang chính thức: [https://nodejs.org/](https://nodejs.org/).
    Sau khi cài đặt Node.js, bạn có thể kiểm tra phiên bản của Node.js và npm bằng cách mở terminal và chạy các lệnh sau:

```bash
node -v
npm -v
```
