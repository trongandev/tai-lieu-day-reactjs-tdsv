# FONT CHỮ

Trong file này, chúng ta sẽ thay đổi font chữ phù hợp khi làm việc với mã nguồn trong Visual Studio Code.

Chúng ta sẽ cài font chữ [`JetBrains Mono`](https://www.jetbrains.com/lp/mono/), một font chữ mã nguồn phổ biến, dễ đọc và hỗ trợ nhiều ký tự đặc biệt.

## Bước 1: Truy cập trang tải font JetBrains Mono

Mở trình duyệt và truy cập trang chính thức của JetBrains Mono tại địa chỉ: [https://www.jetbrains.com/lp/mono/](https://www.jetbrains.com/lp/mono/)

## Bước 2: Tải font chữ

Trên trang JetBrains Mono, tìm và nhấn vào nút "Download" để tải về file nén chứa font chữ.

## Bước 3: Cài đặt font chữ

Giải nén file vừa tải về, sau đó mở thư mục chứa các file font. Chọn tất cả các file font (có đuôi .ttf hoặc .otf), nhấp chuột phải và chọn "Install" để cài đặt font chữ vào hệ thống.

## Bước 4: Cấu hình Visual Studio Code sử dụng font mới

Mở Visual Studio Code, vào phần cài đặt (Settings), tìm kiếm "Font Family" và thay đổi giá trị thành "JetBrains Mono". Nếu muốn, bạn cũng có thể điều chỉnh kích thước font chữ trong phần "Font Size" để phù hợp với mắt nhìn của mình.

## Bước 5: Khởi động lại Visual Studio Code

Đóng và mở lại Visual Studio Code để áp dụng thay đổi font chữ mới.
Bây giờ bạn đã hoàn tất việc thay đổi font chữ trong Visual Studio Code sang JetBrains Mono. Chúc bạn có trải nghiệm lập trình thú vị và dễ chịu hơn với font chữ mới này!
